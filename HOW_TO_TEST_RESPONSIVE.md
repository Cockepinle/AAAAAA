# 📱 Как тестировать адаптивность на разных устройствах (F12 DevTools)

## ❓ Проблема: "F12 ничего не открывает"

**Решение:** Вы жали F12 в PowerShell/CMD терминале, а не в браузере!

---

## ✅ Правильный порядок действий

### Шаг 1️⃣ — Запустите сервер в терминале

```powershell
cd "c:\Users\alepi\OneDrive\Рабочий стол\shopur\shopur"
python manage.py runserver
```

Вы должны увидеть:
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

### Шаг 2️⃣ — Откройте браузер

**Выберите один:**
- Google Chrome
- Microsoft Edge
- Mozilla Firefox

И перейдите на: `http://localhost:8000`

### Шаг 3️⃣ — Откройте DevTools

**Нажмите одну из комбинаций клавиш:**

| Браузер | Горячая клавиша |
|---------|-----------------|
| Chrome | `F12` или `Ctrl+Shift+I` |
| Edge | `F12` или `Ctrl+Shift+I` |
| Firefox | `F12` или `Ctrl+Shift+I` |

**Откроется панель внизу/справа:**
```
┌─────────────────────────────────────────────┐
│                  Elements  Console  Sources  │  ← Вкладки
├─────────────────────────────────────────────┤
│                                              │
│   <html>                                     │
│     <head>...</head>   ← Это HTML код       │
│     <body>...</body>                         │
│                                              │
└─────────────────────────────────────────────┘
```

### Шаг 4️⃣ — Включите мобильный вид

**В панели DevTools найдите иконку телефона:**

```
╔══════════════════════════════════════════════╗
║ ☰   │  🔍 │ >> │ ⚙️  │ ... │  📱  │  🖥️  │  ← Найти иконку телефона
╚══════════════════════════════════════════════╝
```

**Или используйте горячую клавишу:**
- **Chrome/Edge/Firefox:** `Ctrl+Shift+M`

**Или нажмите правой кнопкой мыши на страницу:**
1. Правый клик на странице
2. Выберите `Inspect` или `Инспектировать`
3. Откроется DevTools + автоматически включится мобильный вид

## 🎮 Управление в DevTools

### Изменение размера экрана

**Способ 1: Выпадающее меню**
```
Responsive ▼  │  ☐  375 x 667  │  100%
```
Нажимайте на размер и выбирайте новый

**Способ 2: Ручное введение**
```
375 x 667  ← Нажмите, введите новый размер
```

**Способ 3: Перетаскивание**
```
Возьмитесь за край мобильного окна и тащите мышью
```

### Поворот экрана

```
[📱] ← Портретная ориентация (узкий)
[🔄]  ← Кнопка поворота
[📱] ← Ландшафтная ориентация (широкий)
```

### Масштабирование

```
100%  ← Текущий масштаб
 │
 ├─ 75%
 ├─ 100%  ← Нормальный
 ├─ 125%
 └─ 150%
```

## 🐛 Отладка элементов

### Выделение элемента

```
1. Нажмите Ctrl+Shift+C (или иконку указателя в DevTools)
2. Наведите мышь на элемент на странице
3. Элемент будет выделен и его код появится в DevTools
```

### Изменение CSS на лету

```
1. Найдите элемент в DevTools
2. Справа будет панель с CSS стилями
3. Отредактируйте значение, например:
   width: 100%;  ← Измените на width: 50%;
4. Изменение применится сразу (только в браузере, не в файле)
```

### Просмотр адаптивных классов

```
DevTools → Elements → Найдите элемент
Посмотрите на классы:
<div class="col-12 col-md-6 col-lg-4">
                 ↓        ↓
         Смартфон  Планшет  Десктоп
```

## 💡 Полезные советы

### ✅ Используйте сочетания клавиш

```
F12                    Открыть/закрыть DevTools
Ctrl+Shift+M           Включить/выключить мобильный вид
Ctrl+Shift+C           Выделение элемента на странице
Ctrl+Shift+I           Альтернатива F12
Ctrl+Shift+J           Открыть Console (консоль)
```

### ✅ Смотрите Console на ошибки

```
1. DevTools → Console вкладка
2. Посмотрите красные ошибки
3. Если есть ошибка, скажите мне что там написано
```

### ✅ Очищайте кеш

```
DevTools → Settings (⚙️) → Network conditions
Поставьте галочку "Disable cache"
(работает только когда DevTools открыт)
```

## 📞 Если что-то не работает

**Проверьте:**

- ✅ Сервер запущен (видна строка "Starting development server")
- ✅ Браузер открыт на `http://localhost:8000`
- ✅ F12 нажали в окне браузера, а не в терминале
- ✅ Мобильный вид включён (внизу окна должна быть мобильная сетка)

**Если ничего не помогает:**
1. Закройте все браузеры
2. Нажмите Ctrl+C в терминале (остановит сервер)
3. Запустите заново: `python manage.py runserver`
4. Откройте браузер заново
