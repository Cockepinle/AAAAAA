# 📱 Адаптивный интерфейс (Responsive Design) — Полная документация

## Что такое адаптивный интерфейс?

**Адаптивный интерфейс (Responsive Design)** — это подход в веб-разработке, при котором дизайн и макет сайта автоматически приспосабливаются к размеру экрана пользователя. Вместо создания отдельного сайта для каждого устройства, используется один гибкий код, который перестраивается.

### Преимущества:
- ✅ Один код для всех устройств
- ✅ Лучше для SEO
- ✅ Улучшенный UX на мобильных
- ✅ Экономия на разработке
- ✅ Легче поддерживать

---

## 🎯 Три основных брейкпоинта

Брейкпоинт — это точка разрешения, при которой дизайн переходит на другой вид.

### **Брейкпоинт 1: СМАРТФОН (320px - 767px)**

**Когда применяется:** Мобильные телефоны, портретная ориентация

**Характеристики:**
- Ширина экрана: от 320px до 767px
- 1 колонка контента
- Вертикальное расположение элементов
- Крупные кнопки (45-50px)
- Полная ширина форм
- Бургер-меню вместо полного навигационного меню
- Текст: 14-16px (для удобства)
- Меньше элементов на странице

**Примеры стилей:**
```css
@media (max-width: 767px) {
    h1 { font-size: clamp(1.5rem, 5vw, 2rem); }
    .container { padding-left: 1rem; padding-right: 1rem; }
    .navbar { flex-direction: column; }
    .catalog-grid { grid-template-columns: 1fr; }
    .cart-card { flex-direction: column; }
}
```

**Что включено:**
- Каталог: 1 товар в ряду
- Корзина: вертикальное расположение товаров
- Заказы: вертикальный вид
- Фильтры: свёрнуты (collapse), развернуть по кнопке
- Категории: 2 в ряду (col-6)

---

### **Брейкпоинт 2: ПЛАНШЕТ (768px - 1023px)**

**Когда применяется:** Планшеты (iPad), ландшафтная ориентация

**Характеристики:**
- Ширина экрана: от 768px до 1023px
- 2 колонки контента
- Горизонтальное меню (если поместилось)
- Кнопки среднего размера (40px)
- Формы: 1-2 в ряду
- Текст: 15px
- Оптимальное использование пространства

**Примеры стилей:**
```css
@media (min-width: 768px) and (max-width: 1023px) {
    h1 { font-size: clamp(1.75rem, 5vw, 2.5rem); }
    .container { max-width: 720px; }
    .catalog-grid { grid-template-columns: repeat(2, 1fr); }
    .product-list-item { flex-direction: row; }
}
```

**Что включено:**
- Каталог: 2 товара в ряду
- Корзина: горизонтальное расположение
- Заказы: улучшенный вид
- Категории: 3-4 в ряду (col-md-4)
- Фильтры: видны, но в узком виде

---

### **Брейкпоинт 3: ДЕСКТОП (1024px и выше)**

**Когда применяется:** Компьютеры, ноутбуки, широкие мониторы

**Характеристики:**
- Ширина экрана: 1024px и выше
- 3-4+ колонки контента
- Полное горизонтальное меню
- Кнопки нормального размера (45px)
- Формы: максимум 4 в ряду
- Текст: 16px
- Максимальное использование пространства

**Примеры стилей:**
```css
@media (min-width: 1024px) {
    h1 { font-size: clamp(2.5rem, 6vw, 3.5rem); }
    .container { max-width: 1200px; }
    .catalog-grid { grid-template-columns: repeat(4, 1fr); }
    .mega-menu .menu-columns { column-count: 3; }
}
```

**Что включено:**
- Каталог: 4 товара в ряду
- Корзина: полноценный горизонтальный вид
- Заказы: максимальная информация видна сразу
- Категории: 6+ в ряду
- Фильтры: всегда видны

## 🛠️ Как это реализовано в проекте

### 1. **CSS Media Queries (файл: `users/static/css/style.css`)**

```css
/* СМАРТФОН: 320px - 767px */
@media (max-width: 767px) {
    body { font-size: 14px; }
    .catalog-grid { grid-template-columns: 1fr; gap: 1rem; }
    .product-list-item { flex-direction: column; }
}

/* ПЛАНШЕТ: 768px - 1023px */
@media (min-width: 768px) and (max-width: 1023px) {
    body { font-size: 15px; }
    .catalog-grid { grid-template-columns: repeat(2, 1fr); gap: 1.25rem; }
}

/* ДЕСКТОП: 1024px+ */
@media (min-width: 1024px) {
    body { font-size: 16px; }
    .catalog-grid { grid-template-columns: repeat(4, 1fr); gap: 1.5rem; }
}
```

### 2. **Viewport Meta Tag (в base.html)**

```html
<meta name="viewport" content="width=device-width, initial-scale=1">
```

Это **обязательно** для мобильных устройств!

### 3. **Bootstrap Grid System**

```html
<!-- 1 на мобиле, 2 на планшете, 4 на десктопе -->
<div class="row row-cols-1 row-cols-sm-2 row-cols-lg-4 g-3">
    <div class="col">...</div>
</div>
```

**Объяснение:**
- `row-cols-1` — 1 колонка по умолчанию (мобиль)
- `row-cols-sm-2` — 2 колонки с ширины 576px
- `row-cols-lg-4` — 4 колонки с ширины 992px

### 4. **Гибкие размеры (CSS Functions)**

```css
/* Размер автоматически масштабируется */
h1 { font-size: clamp(1.5rem, 5vw, 2rem); }
/* Минимум 1.5rem, максимум 2rem, пропорционально ширине */
```

### 5. **Гибкие изображения**

```html
<!-- Занимает 100% ширины контейнера -->
<img src="photo.jpg" class="img-fluid" loading="lazy" alt="Описание">
```

## 🎨 Стили для адаптивности

### Utility Classes (готовые классы)

```css
/* Скрывать элементы по размеру */
.hide-mobile { display: none; }      /* Показать только на планшете+ */
.hide-tablet { display: block; }     /* Показать на мобиле и десктопе */
.hide-desktop { display: block; }    /* Скрыть только на десктопе */

/* Используются так: */
<div class="hide-mobile">Видно только на планшете и десктопе</div>
<div class="hide-desktop">Видно только на мобиле и планшете</div>
```

### Гибкие компоненты

```html
<!-- Адаптивный flex: вертикально на мобиле, горизонтально на планшете+ -->
<div class="d-flex flex-column flex-md-row gap-2">
    <button class="btn btn-gold w-100 w-md-auto">Кнопка</button>
</div>

<!-- Разные отступы для разных размеров -->
<div class="p-2 p-md-3 p-lg-4">Контент</div>

<!-- Разные размеры шрифта -->
<h1 class="fs-3 fs-md-2 fs-lg-1">Заголовок</h1>
```

## 🔍 Как тестировать адаптивность

### Способ 1: DevTools в браузере
1. Откройте Chrome/Firefox
2. Нажмите `F12` (открыть DevTools)
3. Нажмите `Ctrl+Shift+M` (мобильный вид)
4. Выберите устройство: iPhone, iPad, Pixel и т.д.
5. Смотрите как меняется макет при изменении размера

### Способ 2: Экран браузера
1. Откройте сайт
2. Перетаскивайте край окна браузера
3. Смотрите как перестраивается макет

### Способ 3: Реальное устройство
1. На рабочем столе откройте командную строку
2. Запустите: `python manage.py runserver 0.0.0.0:8000`
3. На телефоне откройте: `http://<IP_КОМПЬЮТЕРА>:8000`
4. Проверьте как выглядит на реальном экране

## 💡 Лучшие практики

### ✅ ДЕЛАЙТЕ:

```html
<!-- 1. Мобильный подход (Mobile First) -->
<style>
    .product-card { width: 100%; }  /* По умолчанию 100% -->
    
    @media (min-width: 768px) {
        .product-card { width: 48%; }  /* На планшете 48% -->
    }
    
    @media (min-width: 1024px) {
        .product-card { width: 23%; }  /* На десктопе 23% */
    }
</style>

<!-- 2. Используйте flexbox -->
<div style="display: flex; flex-wrap: wrap; gap: 1rem;">
    <div>Элемент 1</div>
    <div>Элемент 2</div>
</div>

<!-- 3. Используйте viewport meta tag -->
<meta name="viewport" content="width=device-width, initial-scale=1">

<!-- 4. Оптимизируйте изображения -->
<img src="photo.jpg" class="img-fluid" loading="lazy" alt="Описание">
```

### ❌ НЕ ДЕЛАЙТЕ:

```html
<!-- ❌ Жёсткие размеры -->
<div style="width: 800px;">Контент</div>

<!-- ❌ Без viewport метатега -->
<!-- (сайт будет масштабироваться неправильно) -->

<!-- ❌ Огромные изображения -->
<img src="photo.jpg" width="2000" height="2000">

<!-- ❌ Усложнённые медиа-запросы -->
@media (min-width: 320px) and (max-width: 480px) and (orientation: portrait) { ... }
```
