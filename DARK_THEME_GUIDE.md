## 🎨 Тестирование темной темы

### Способ 1: Через админ-панель (если реализовано)
1. Заходите в профиль
2. Редактируете профиль
3. Выбираете тему: **Dark**
4. Сохраняете

### Способ 2: Добавить класс `dark` к body (для теста)
```html
<!-- В base.html измените: -->
<body class="dark">  <!-- Было: class="light" -->
```

### Способ 3: Через DevTools
1. Откройте DevTools (F12)
2. В Console введите:
```javascript
document.body.classList.add('dark');
// или
document.body.classList.toggle('dark');
```
